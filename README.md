# Air Traffic Scheduling System

This is a mini project developed in C during my second semester. The project simulates the basic operations of an airport by managing flight schedules, runway allocation, crew assignment and emergency situations. Flight and crew details are stored using a MySQL database.

## Features

- Add new flights
- Display flight schedule
- Assign crew members
- Allocate runways
- Handle emergency situations
- Delay or cancel flights
- Generate runway utilization reports
- Audio alerts for announcements

## Technologies Used

- C Programming
- MySQL
- SQL
- Windows API
- Visual Studio Code

## Files

- `ATSS3.c` - Main source code
- `announcement-sound-104411.wav` - Announcement sound
- `international-airport-intercom-madeira-19947.wav` - Airport sound

## How to Run

1. Install MySQL Server.
2. Create the required database and tables.
3. Update the MySQL username and password in the source code if needed.
4. Compile the program.
5. Run the application.
